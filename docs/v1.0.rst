Overview
--------

This project provides a new family of collections.

Installation
------------

To install ``namings``, you can use ``pip``.
Open your terminal and run:

.. code-block:: shell

    pip install namings

``__contains__(other: Any) -> bool``
""""""""""""""""""""""""""""""""""""

Here, it establishes a standard for all descendent classes.
It should only ever be overwritten with an extensional equivalent.

    def __contains__(self: Self, other: Any, /) -> bool:
        return other in self.items()

``get(key: object, default: Any = None, /) -> Any``
"""""""""""""""""""""""""""""""""""""""""""""""""""

This method returns the value under the given ``key`` if present,
otherwise it returns ``default``
Here, it establishes a standard for all descendent classes.
It should only ever be overwritten with an extensional equivalent.

.. code-block:: python

    def get(self: Self, key: object, default: Any = None, /) -> Any:
        "This method returns the value for an existing key or default for a not existing key."
        try:
            return self[key]
        except KeyError:
            return default

``namings.core``
^^^^^^^^^^^^^^^^

The classes in this submodule can also be imported directly from ``namings``.

``class namings.core.BaseNaming.BaseNaming(data: Any = (), /, **kwargs: Any)``
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

This abstract base class is a common ancestor of ``FrozenNaming`` and ``Naming``.

``class namings.core.FrozenNaming.FrozenNaming(data: Any = (), /, **kwargs: Any)``
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

This immutable class is a type of collection

- that coerces keys to be strings
- and that acts similarly to a sequence of its items.

``class namings.core.Naming.Naming(data: Any = (), /, **kwargs: Any)``
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

This mutable class is a type of collection

- that coerces keys to be strings
- and that acts similarly to a sequence of its items.
